# 💈 Garagem do Gueto

Sistema de agendamento para barbearia desenvolvido com HTML, CSS e JavaScript, com foco em uma interface moderna, responsiva e intuitiva.

## 🚀 Demonstração

🔗 https://garagem-do-gueto.vercel.app

## 📸 Preview

> Adicione aqui um print da aplicação.

## ✨ Funcionalidades

- Agendamento de serviços
- Seleção de barbeiro
- Escolha de data e horário
- Interface responsiva
- Design moderno
- Navegação simples e intuitiva

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Estrutura do projeto

```
📦 garagem-do-gueto
 ├── index.html
 ├── style.css
 ├── script.js
 └── README.md
```

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/uJ-Desenvolvedor/barbearia.git
```

Entre na pasta:

```bash
cd barbearia
```

Depois basta abrir o arquivo `index.html` no navegador.

## 🌐 Deploy

A aplicação está hospedada na Vercel.

Sempre que um novo commit é enviado para a branch `main`, a Vercel realiza um novo deploy automaticamente.
